# gary-travel-site

Next.js App Router 專案骨架，已啟用 TypeScript 與 Tailwind CSS，適合直接部署到 Vercel。

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## Scripts

```bash
npm install
npm run dev
```

其他指令：

```bash
npm run build
npm run start
npm run lint
npm run type-check
```

## Structure

```text
src/
  app/
    globals.css
    layout.tsx
    page.tsx
  components/
  lib/
```
